# 统一数据库快速操作手册

## 1. 启动后端（数据统一核心）
在你的电脑执行：

```powershell
cd D:\github\50！\diet-h5\backend
npm install
npm start
```

启动后访问：
- 本机：`http://localhost:3000`
- 手机：`http://<你的电脑局域网IP>:3000`

## 2. 查电脑局域网 IP
```powershell
ipconfig
```
在 `无线局域网适配器 Wi-Fi` 中找到 `IPv4 地址`，例如 `192.168.1.23`。

## 3. 防火墙放行 3000 端口（如果手机访问不到）
允许 Node 或 3000 端口入站。

## 4. 数据一致性验证
1. 手机端新增一条今日记录。
2. 电脑端打开后台记录列表刷新。
3. 应看到同一条数据。

## 5. 注意
- 如果仍用 `file://` 打开的页面，仍可能走本地存储。
- 必须通过后端地址访问（`http://...:3000`）才能走统一数据库。
