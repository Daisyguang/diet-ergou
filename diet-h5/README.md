# Diet H5 Prototype

## 1) 安装依赖
```powershell
cd D:\github\50！\diet-h5\backend
npm install
```

## 2) 启动
```powershell
npm start
```

## 3) 打开
浏览器访问: http://localhost:3000

## 功能
- 今日记录（三餐图片最多3张+文字）
- 三项目标打卡
- 体重多次记录（当天最后一次用于掉秤）
- 生成三餐汇总长图并保存到本地数据目录
- 历史查看
- 设置开始记录日
